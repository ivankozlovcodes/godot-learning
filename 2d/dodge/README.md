# Dodge the Creeps

Learning project — Godot 4 port of the official [Dodge the Creeps](https://github.com/godotengine/godot-demo-projects/tree/master/2d/dodge_the_creeps) demo.

## What it is

2D survival game. Player dodges enemies (creeps) that spawn from screen edges and move in random directions. Score = time survived. Game ends on collision.

## Controls

| Action | Key / Button |
|--------|-------------|
| Move | Arrow keys |
| Start | Space / Gamepad A |

Gamepad supported.

## Stack

- Godot 4.6
- GDScript
- 480×720 viewport (portrait)

## Run

Open `project.godot` in Godot 4.6+, press F5.
