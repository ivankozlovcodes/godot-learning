# Breakout

A Breakout clone built in Godot 4.

## Assets

Sound effects and music: [Brackeys Platformer Assets](https://brackeysgames.itch.io/brackeys-platformer-bundle) (CC0)

- Sounds by Brackeys, Asbjørn Thirslund
- Music by Brackeys, Sofia Thirslund
- Sprites by [analogStudios_](https://analogstudios.itch.io/) and [RottingPixels](https://rottingpixels.itch.io/)
- Fonts by Jayvee Enaguas (HarvettFox96) — [Pixel Operator](https://www.dafont.com/pixel-operator.font)
- Bonus sound by [Universfield](https://pixabay.com/users/universfield-28281460/) via Pixabay
- Additional SFX from [Ultimate 8bit SFX Library Vol 3](https://ef9.itch.io/ultimate-8bit-sfx-library-vol-3) by ef9
